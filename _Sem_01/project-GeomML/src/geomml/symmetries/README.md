# symmetries

## Submodule for symmetries.

`handcrafted.py` cодержит кастомные симмерии, например:
* rotate_phi(...)
* swap_jets(...)
* reflection(...)

`atlas.py` реализует:
* AtlasDiscovery()
* AtlasSampler()
* OrbitGenerator()

`lie_group.py` содержит
* SO(2)
* SO(3)
* SE(3)

или их приближения.
